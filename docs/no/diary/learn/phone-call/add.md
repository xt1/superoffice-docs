# Registrere samtaler

1. Gjør ett av følgende:

    * Velg **Ny** > **Samtale** på [toppraden][2].
    * Klikk **på Oppgave** ( ![Ikonet][img1] ) > **Opprett samtale** i et personkort.

1. I dialogboksen **Samtale** , [angi detaljene for samtalen][1].
